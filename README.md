# Financial Performance Dashboard

## Project Overview

This project presents an interactive **Financial Performance Dashboard** developed using **Microsoft Power BI** to analyze an organization's financial data across multiple years and countries.

The dashboard transforms General Ledger (GL) data into meaningful financial insights by visualizing key financial statements, profitability metrics, revenue trends, and country-level performance indicators.

The objective of this project is to demonstrate practical **Financial Analyst** skills including financial reporting, profitability analysis, KPI monitoring, trend analysis, and interactive dashboard development.

---

## Project Objectives

- Analyze company financial performance across multiple years.
- Monitor revenue growth and profitability.
- Prepare and visualize Profit & Loss Statements.
- Compare financial performance across countries.
- Track key profitability metrics and financial ratios.
- Enable dynamic filtering and business analysis through interactive dashboards.
- Support data-driven decision-making.
