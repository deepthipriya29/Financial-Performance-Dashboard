# Financial-Performance-Dashboard
Project Overview
This project presents an interactive Financial Performance Dashboard developed using Microsoft Power BI to analyze an organization's financial data across multiple years and countries. The dashboard transforms General Ledger (GL) data into meaningful financial insights by visualizing key financial statements, profitability metrics, and business performance indicators.
The objective of this project is to demonstrate practical Financial Analyst skills including financial reporting, profitability analysis, trend analysis, KPI monitoring, and interactive dashboard development.
Project Objectives
Analyze company financial performance over multiple years.
Monitor revenue and profitability using interactive dashboards.
Prepare and visualize Profit & Loss statements.
Compare financial performance across countries.
Track key profitability metrics and financial ratios.
Enable business users to filter and analyze data dynamically.
